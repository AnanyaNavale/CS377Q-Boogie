# Boogie

A ride-share prototype designed for blind and low-vision (BLV) users – screen reader-friendly, map-free, and universally usable.

Built for CS 377Q: Designing for Accessibility at Stanford University, Winter 2026.

---

## About

Boogie reimagines the ride-share experience without relying on visual maps. Instead of the typical map-based pickup and drop-off flow, Boogie uses a landmark-based location system and Open AI-supported chatbot that works seamlessly with screen readers.

Features include:
- Map-free pickup and drop-off selection
- Full screen reader compatibility
- Open AI-based chatbot for conversational information access
- Images throughout for universal usability
- Color mode toggle for light-sensitive low-vision users

## Run Locally

1. Clone the repository
```
git clone https://github.com/AnanyaNavale/CS377Q-Boogie.git
```

2. Install dependencies
```
npm install
```

3. Start the development server
```
npx expo start
```

Then scan the QR code in your terminal with Expo Go.
